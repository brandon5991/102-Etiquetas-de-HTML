102 Etiquetas de HTML

Nombre de la Etiqueta Descripción Ejemplo de Cómo Usar Cualquier Elemento Adicional
<a> Define un hipervínculo <a href="https://www.example.com">Enlace</a> El atributo href especifica la URL del enlace
<b> Define texto en negrita <b>Texto en negrita</b> Este elemento no es semántico, solo da formato
<br> Inserta un salto de línea <p>Texto antes<br>Texto después</p> No tiene etiqueta de cierre
<div> Define una sección de bloque <div>Contenido</div> Utilizado para agrupar elementos
<em> Define texto enfatizado <em>Texto enfatizado</em> Generalmente se muestra en cursiva
<form> Define un formulario <form action="/submit" method="post"></form> Los formularios contienen campos para la entrada del usuario
<h1> Define el título principal <h1>Título</h1> Hay seis niveles de encabezado (<h1> a <h6>)
<img> Define una imagen <img src="imagen.jpg" alt="Descripción"> No tiene etiqueta de cierre
<input> Define un control de entrada <input type="text" name="nombre"> Hay diferentes tipos de controles, como texto, radio, etc.
<li> Define un elemento en una lista <ul><li>Elemento</li></ul> Se utiliza dentro de listas no ordenadas (<ul>) y ordenadas (<ol>)
<meta> Define metadatos para el documento <meta charset="UTF-8"> Los metadatos proporcionan información sobre el documento
<ol> Define una lista ordenada <ol><li>Elemento 1</li><li>Elemento 2</li></ol> Los elementos se numeran automáticamente
<p> Define un párrafo <p>Texto del párrafo</p> Los párrafos se separan por márgenes
<span> Define una sección en línea <span>Texto</span> Se utiliza para aplicar estilos o scripts a partes del texto
<strong> Define texto con importancia <strong>Texto importante</strong> Generalmente se muestra en negrita
<table> Define una tabla <table><tr><td>Celda</td></tr></table> Las tablas están formadas por filas (<tr>) y celdas (<td>)
<ul> Define una lista no ordenada <ul><li>Elemento</li></ul> Los elementos se muestran con viñetas
<video> Define un video <video src="video.mp4" controls></video> Puede incluir controles de reproducción
<footer> Define el pie de página <footer>Contenido del pie de página</footer> Se utiliza para información de contacto y derechos de autor
<header> Define el encabezado de un documento o sección <header>Contenido del encabezado</header> Puede contener títulos, logotipos y menús
<article> Define un artículo independiente <article>Contenido del artículo</article> Se utiliza para contenido autónomo
<section> Define una sección de un documento <section>Contenido de la sección</section> Utilizado para agrupar contenido temáticamente relacionado
<aside> Define contenido complementario <aside>Contenido adicional</aside> Generalmente se utiliza para barras laterales
<nav> Define enlaces de navegación <nav><a href="home.html">Inicio</a></nav> Usado para agrupar enlaces de navegación
<details> Define detalles adicionales <details><summary>Título</summary>Contenido</details> Permite al usuario mostrar u ocultar detalles
<summary> Define un resumen para un elemento <details> <details><summary>Título</summary>Contenido</details> Debido a <details>, el contenido dentro de <summary> se muestra primero
<main> Define el contenido principal del documento <main>Contenido principal</main> Debe contener el contenido único del documento
<mark> Define texto resaltado <mark>Texto resaltado</mark> Generalmente se muestra con un fondo amarillo
<progress> Define una barra de progreso <progress value="50" max="100"></progress> Utilizado para mostrar el progreso de una tarea
<meter> Define una medición en un rango conocido <meter value="70" min="0" max="100"></meter> Utilizado para representar un valor dentro de un rango
<time> Define una fecha u hora <time datetime="2024-09-09">9 de septiembre de 2024</time> Puede ayudar a los motores de búsqueda a interpretar el contenido temporal
<figure> Define contenido independiente <figure><img src="imagen.jpg" alt="Descripción"><figcaption>Descripción</figcaption></figure> Se utiliza para agrupar imágenes, ilustraciones y sus leyendas
<figcaption> Define una leyenda para un elemento <figure> <figure><img src="imagen.jpg" alt="Descripción"><figcaption>Descripción</figcaption></figure> Proporciona una descripción para el contenido de <figure>
<dialog> Define un cuadro de diálogo o ventana emergente <dialog open>Contenido</dialog> Puede tener el atributo `open` para mostrar el diálogo
<source> Define múltiples recursos para elementos multimedia <video><source src="video.mp4" type="video/mp4"><source src="video.webm" type="video/webm"></video> Permite especificar diferentes formatos de un recurso multimedia
<template> Define contenido que se puede clonar <template><p>Contenido</p></template> El contenido dentro de <template> no se muestra en la página hasta que se clona
<datalist> Define una lista de opciones para un elemento <input> <input list="browsers"><datalist id="browsers"><option value="Chrome"><option value="Firefox"></datalist> Utilizado para proporcionar sugerencias de entrada
<details> Define detalles adicionales <details><summary>Título</summary>Contenido</details> Permite al usuario mostrar u ocultar detalles
<summary> Define un resumen para un elemento <details> <details><summary>Título</summary>Contenido</details> Debido a <details>, el contenido dentro de <summary> se muestra primero
<data> Define un valor de datos con una unidad de medida <data value="100" datetime="2024-09-09">100</data> El atributo `value` se utiliza para proporcionar el valor numérico
<base> Define una URL base para todos los enlaces relativos en un documento <base href="https://www.example.com/"> Debe estar dentro del elemento <head>
<object> Define un objeto en un documento <object data="file.pdf" type="application/pdf">No se puede mostrar</object> Puede ser utilizado para incrustar diferentes tipos de contenido
<embed> Define un contenedor para contenido externo <embed src="documento.pdf" type="application/pdf"> No tiene etiqueta de cierre
<iframe> Define un marco dentro del documento <iframe src="https://www.example.com" width="600" height="400"></iframe> Utilizado para incrustar otro documento HTML
<script> Define un bloque de código JavaScript <script>alert('Hola Mundo');</script> Puede estar en el <head> o antes del </body>
<style> Define un bloque de CSS <style>body { color: red; }</style> Debe estar dentro del elemento <head>
<link> Define una relación entre el documento y un recurso externo <link rel="stylesheet" href="estilos.css"> Utilizado para vincular hojas de estilo externas
<meta> Define metadatos para el documento <meta charset="UTF-8"> Los metadatos proporcionan información sobre el documento
<noscript> Define un contenido alternativo para navegadores que no soportan JavaScript <noscript>JavaScript no está habilitado</noscript> Se muestra cuando JavaScript está deshabilitado o no soportado
<svg> Define gráficos vectoriales SVG <svg width="100" height="100"><circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red"/></svg> Utilizado para gráficos escalables
<math> Define contenido matemático <math><msup><mi>x</mi><mn>2</mn></msup></math> Permite insertar notación matemática en HTML
<address> Define información de contacto <address>123 Calle Principal, Ciudad</address> Usado para información de contacto del autor o del sitio
<cite> Define una referencia a una fuente <cite>Autor del libro</cite> Usado para citar una obra, como un libro o un artículo
<q> Define una cita corta <q>Texto de la cita</q> Generalmente se muestra entre comillas
<kbd> Define la entrada del teclado <kbd>Ctrl+S</kbd> Se utiliza para representar la entrada del teclado
<dfn> Define un término que está siendo definido <dfn>HTML</dfn> Usado para términos definidos en un documento
<var> Define una variable en un documento <var>x</var> Se utiliza para variables en matemáticas y programación
<code> Define un fragmento de código <code>var x = 10;</code> Usado para mostrar código fuente en línea
<pre> Define texto preformateado <pre>Texto en formato preestablecido</pre> Conserva espacios y saltos de línea
<bdi> Define una secuencia de texto que debe ser aislada del texto circundante <bdi>Texto en otro idioma</bdi> Se utiliza para evitar que el texto sea afectado por el formato del texto circundante
<bdo> Define la dirección del texto <bdo dir="rtl">Texto de derecha a izquierda</bdo> Se utiliza para cambiar la dirección del texto
<ruby> Define una anotación de ruby <ruby>漢字<rt>Kanji</rt></ruby> Se utiliza para mostrar anotaciones sobre caracteres
<rt> Define la anotación para un elemento <ruby> <ruby>漢字<rt>Kanji</rt></ruby> Se utiliza para mostrar la pronunciación de los caracteres
<rp> Define lo que se mostrará en navegadores que no soportan <ruby> <ruby>漢字<rp>(漢字)</rp></ruby> Se utiliza como una alternativa en caso de que <ruby> no sea soportado
